დეკლარაცია (Declaration):
CSS-ში დეკლარაცია არის წყვილი, რომელიც აერთიანებს სტილს

კუთვნილება (Property):
ეს არის სტილის თვისება, რომელსაც უკავშირდება მნიშვნელობა

მნიშვნელობა (Value):
მნიშვნელობა არის ის, რაც სტილის თვისებას ენიჭება

CSS Rule (CSS წესი):
CSS წესი შედგება selector-ისა და დეკლარაციის ბლოკისგან
